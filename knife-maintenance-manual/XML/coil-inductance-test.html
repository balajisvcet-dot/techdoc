Coil Inductance Test

  1. Perform Safety Lockout procedure in Operation section.
  2. Each cylinder motor must be tested individually using the following steps.
  3. Disconnect all the motor armature wires (Phases A, B, C, D) from the associated cylinder drive.
  4. Turn on the LCR meter and set it to Inductance/Capacitance measuring mode, 20mH scale.
  5. All the phases must be tested individually. Connect the LCR meter leads to the wires of each phase as detailed below. Polarity is not critical, but allow the meter to stabilize and then record the result.
    1. Phase A--motor wires A1,A2 _________________________________________________________________________
    2. Phase B--motor wires B1, B2 _________________________________________________________________________
    3. Phase C--motor wires C1, C2 _________________________________________________________________________
    4. Phase D--motor wires D1, D2 _________________________________________________________________________
  6. Refer to the table below for the coil inductance values and tolerance range per the BW Papersystems part number. (The motor number can be found on the motor nameplate.)
  7. If any of the measurements do not fall within the acceptable tolerance, record the observed measurement, which phase, motor part #, motor serial #, and motor position. Replace the failed motor as soon as possible and return it to BW Papersystems with the detailed information recorded above. Contact BW Papersystems Field Service for assistance.
  8. Reconnect the motor armature wires to the drive per print.
